A quick tutorial...

Access the app

Press CTRL + ALT + L to bring up the mini editor.
Press CTRL + ALT + L again to make it big.
Press CTRL + ALT + K to bring up the big editor right away.

Create Logs and Categories

Type a note and press Enter to create a log in the default Notes section.
Type a / followed by a word to create a category e.g. /pickles to create a category for notes about pickles.
Optionally, type a note after a new category name to create a log in that category at the point of creation.

Navigate Categories and Logs

Press Tab / Shift + Tab or Right Arrow / Left Arrow to move between categories.
When a Category is selected press Up Arrow / Down Arrow to select a log.

Manage Logs

When a log is selected press Enter to mark it as done.
Alternatively press Delete to delete a selected log.

Category bonus features

With a category selected type in a colon to make a sub category e.g. /pickles:recipe
With a category selected type in /e and press Enter to delete all notes in a category e.g. /pickles /e
With a category selected type in /d and press Enter to delete the category e.g. /pickles /d
With a category selected type in /m# to move the position of the category relative to notes e.g. /pickles /m1 will move a category one to the right of Notes.
