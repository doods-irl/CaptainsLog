A quick tutorial...

Right click on the Captain's Log icon in the notification tray and select Tutorial to find this page again.

Access the app

Press CTRL + ALT + L to bring up the mini editor.
Press CTRL + ALT + L again to make it big.
Press CTRL + ALT + K to bring up the big editor right away.
Press Escape to hide the editor.

Create Logs and Categories

Type a note and press Enter to create a log in the default Notes section.
Type a / followed by a word to create a category, for example /pickles.
Optionally, type a note after a new category name to create a log in that category as it's created, for example /pickles Taste just okay...
Press Space after an existing category name, for example /pickles , to target that category for follow-up commands or a new note.

Navigate Categories and Logs

With the log entry box focused:
Press Tab / Shift + Tab or Right Arrow / Left Arrow to move between categories.
Type a / followed by a category name to filter categories, then press Tab until you've selected the one you're after.
Press Up Arrow / Down Arrow to select a log.
Press Shift + Up Arrow / Shift + Down Arrow to jump back to the log entry box or select the last log in the category.

Manage Logs

When a log is selected:
Start typing to edit its contents.
Press Shift + Enter to mark it as done.
Press Shift + Delete to delete the log.

Category advanced features

With a category selected:
Type a colon to make a sub-category, for example /pickles:recipe.
Sub-categories are hidden by default; adding a colon after any category name will display its sub-categories in the nav bar.
Type /e and press Enter twice to empty a category, for example /pickles /e.
Type /d and press Enter twice to delete a category, for example /pickles /d.
Alternatively, empty: or delete: followed by a category name will do the same thing immediately, for example empty:pickles.
Type /m# to move the position of the category relative to the Notes category, for example /pickles /m1.

Plugins

Type a dash, for example -, to switch the nav bar from categories to plugin suggestions.
Type part of a plugin name such as -calc or -pom, then use Tab / Shift + Tab or Right Arrow / Left Arrow to cycle matching plugins.
Once a plugin is selected, the lower pane becomes that plugin's answer area. Clearing the bar returns you to Notes.
Examples: -calculator 3+3, -pomodoro 25, -conv 10 km mi, -text slug Captain's Log, -rx /cat/gi :: The Cat scattered a catalog of cats.
Some text-output plugins can act like quick tools: for example -uuid 3 or -transform slug some title then press Enter to copy the result and clear the bar.

Settings

Right click on the Captain's Log icon in the notification tray and select Settings.
From there you can quickly access your config, log files, and set the color theme of the editor.
